# 🧪 خطوات الاختبار — منصة نظامي

> شغّل المشروع بـ `npm run dev` وافتح `http://localhost:3000` في متصفح خاص (Incognito)

---

## ١. الصفحة الرئيسية
- [ ] افتح الصفحة الرئيسية → لازم تشوف الهيرو والخدمات والإحصائيات والمؤسسين والفوتر
- [ ] الهيدر فيه: لوجو + روابط + زر **دخول** + زر **تسجيل**
- [ ] على الموبايل: القائمة الجانبية فيها أزرار **دخول** و**تسجيل**
- [ ] جرب غيّر اللغة عربي ↔ إنجليزي
- [ ] جرب الوضع الليلي 🌙

## ٢. الدول
- [ ] في الهيدر → الدول المتاحة **السعودية 🇸🇦** و**مصر 🇪🇬** بس
- [ ] الافتراضي هو السعودية

## ٣. تسجيل حساب فرد (طالب خدمة)
- [ ] روح `/signup`
- [ ] اختار السعودية → "أحتاج خدمات قانونية" → "فرد"
- [ ] اكتب الاسم والايميل والباسورد → سجّل
- [ ] لازم تظهر رسالة نجاح
- [ ] تأكد من جدول `profiles` في Supabase: `account_cat=seeker` و `s_type=individual`

## ٤. تسجيل دخول الفرد
- [ ] روح `/login` → ادخل بيانات حساب الفرد
- [ ] لازم يوديك على `/dashboard/seeker/individual`
- [ ] لازم تشوف: الشريط الجانبي + رسالة ترحيب + إحصائيات + زر "إنشاء طلب جديد"

## ٥. إنشاء طلب خدمة
- [ ] من لوحة تحكم الفرد → اضغط **"إنشاء طلب جديد"**
- [ ] اختار نوع الخدمة (مثلاً: استشارة قانونية)
- [ ] اكتب عنوان الطلب → اضغط **"إرسال الطلب"**
- [ ] لازم يظهر الطلب في **"قضاياي"** ومعاه لون نوع الخدمة

## ٦. الشريط الجانبي — لوحة الفرد
- [ ] اضغط **"المستشار الذكي"** → تظهر صفحة AI Chat (قريباً)
- [ ] اضغط **"تصفح المحامين"** → تظهر صفحة Browse (قريباً)
- [ ] اضغط **"قضاياي"** → تظهر قائمة الطلبات
- [ ] اضغط **"المحفظة"** → تظهر صفحة Wallet (قريباً)
- [ ] اضغط **"الرئيسية"** → يرجع للوحة الرئيسية

## ٧. تسجيل حساب محامي (مقدم خدمة)
- [ ] اعمل تسجيل خروج أولاً
- [ ] روح `/signup` بايميل مختلف
- [ ] اختار السعودية → "أقدم خدمات قانونية" → "محامي مستقل"
- [ ] سجّل → تأكد من Supabase: `account_cat=provider` و `p_type=independent_lawyer`

## ٨. لوحة تحكم المحامي
- [ ] سجّل دخول بحساب المحامي → يروح `/dashboard/provider/independent_lawyer`
- [ ] الشريط الجانبي فيه: الرئيسية / الطلبات / لوحة المهام / التقويم / القوالب / المجتمع / المحفظة
- [ ] اضغط **"الطلبات"** → لازم يظهر الطلب اللي عملته في خطوة ٥
- [ ] اضغط **"لوحة المهام"** → لوحة Kanban
- [ ] اضغط **"المجتمع"** → أسئلة المجتمع القانوني
- [ ] اضغط **"التقويم"** و **"القوالب"** و **"المحفظة"** → كل واحد يفتح صفحته

## ٩. الأدوار التانية (اختبار كل واحد على حده)

### شركة محاماة (`law_firm`)
- [ ] سجّل بحساب جديد: "أقدم خدمات" → "شركة محاماة"
- [ ] الشريط الجانبي: الرئيسية / الطلبات / سير العمل / الفريق / التحليلات / المجتمع / المحفظة
- [ ] كل زر في الشريط الجانبي لازم يفتح صفحة (مش 404)

### محامي متدرب (`trainee_lawyer`)
- [ ] سجّل بحساب جديد: "أقدم خدمات" → "محامي متدرب"
- [ ] الشريط الجانبي: الرئيسية / المهام / الصياغة / الأكاديمية / الأداء
- [ ] **"الصياغة"** → فيها مساحة كتابة + زر "طلب اعتماد" (مفيش زر إرسال للعميل)
- [ ] **"الأكاديمية"** → قائمة دورات تعليمية

### موثّق (`notary`)
- [ ] سجّل بحساب جديد: "أقدم خدمات" → "موثّق"
- [ ] الشريط الجانبي: الرئيسية / الرادار / الطابور / القوالب / الأرشيف / المحفظة
- [ ] **"القوالب"** → 7 أنواع وثائق (وكالة عامة، وكالة خاصة، إلخ)
- [ ] **"الرادار"** → خريطة تفاعلية (قريباً) + قائمة الطلبات

### مأذون (`marriage_official`)
- [ ] سجّل بحساب جديد: "أقدم خدمات" → "مأذون"
- [ ] الشريط الجانبي: الرئيسية / الملف الرقمي / وضع المراسم / التقويم / الأرشيف
- [ ] **"وضع المراسم"** → واجهة خضراء بأزرار كبيرة (بيانات الأطراف / المهر / التوقيعات / إتمام)
- [ ] **"الملف الرقمي"** → قائمة متطلبات (هوية الزوج، الفحص الطبي، إلخ)

### محكّم (`arbitrator`)
- [ ] سجّل بحساب جديد: "أقدم خدمات" → "محكّم"
- [ ] الشريط الجانبي: الرئيسية / المحكمة / الأدلة / الأحكام / الضمان / الأرشيف
- [ ] **"الأحكام"** → فورم صياغة حكم (الديباجة / الوقائع / الأسباب / المنطوق)
- [ ] **"الأدلة"** → غرفة أدلة مشفّرة

### شركة (`company`)
- [ ] سجّل بحساب جديد: "أحتاج خدمات" → "شركة"
- [ ] الشريط الجانبي: الرئيسية / فحص العقود / لوحة المهام / المخاطر / الفريق / المحفظة
- [ ] **"فحص العقود"** → منطقة رفع ملفات + نتائج الفحص (قريباً)
- [ ] **"لوحة المهام"** → Kanban Board

### جهة حكومية (`government`)
- [ ] سجّل بحساب جديد: "أحتاج خدمات" → "جهة حكومية"
- [ ] الشريط الجانبي: الرئيسية / الامتثال / القضايا / التقارير
- [ ] **"الامتثال"** → محرك بحث في الأنظمة + فلاتر

### جمعية (`ngo`)
- [ ] سجّل بحساب جديد: "أحتاج خدمات" → "جمعية"
- [ ] الشريط الجانبي: الرئيسية / الامتثال / مجلس الإدارة / المنح
- [ ] **"الامتثال"** → قائمة تحقق (ترخيص ساري ✓ / تقرير مالي ✗ / إلخ)

## ١٠. تسجيل الخروج
- [ ] اضغط زر الخروج 🚪 (الأيقونة الحمراء في الشريط العلوي)
- [ ] لازم يرجعك للصفحة الرئيسية
- [ ] الهيدر يرجع يعرض "دخول" و "تسجيل" بدل "لوحة التحكم"

## ١١. صفحات عامة
- [ ] `/blog` → صفحة المدونة تفتح
- [ ] `/faq` → صفحة الأسئلة الشائعة تفتح
- [ ] `/contact` → صفحة التواصل تفتح
- [ ] `/login` مباشرة من عنوان المتصفح → تفتح (مش 404) — يحتاج إعداد Nginx

---

## ⚠️ ملاحظات
- لو ظهر في Console تحذير `AbortError` → تجاهله، ده طبيعي في وضع التطوير
- لو الصفحة فاضية → جرب Ctrl+Shift+R (تحديث كامل)
- لو فيه مشكلة اتصال Supabase → انتظر شوية وحدّث الصفحة
- على السيرفر: لازم تضيف `try_files $uri $uri/ /index.html;` في إعدادات Nginx
